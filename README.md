# Life Is Good Exam

### Getting started

Clone this project 

via SSH

```Console
$ git clone git@github.com:jkbicbic/Life-is-good-exam.git
```

via HTTPS

```Console
$ git clone https://github.com/jkbicbic/Life-is-good-exam.git
```

Install dependencies

```Console
$ npm install
```

Starting local server

```Console
$ npm start or yarn start
```

View it in browser

```
http://localhost:3000
```
